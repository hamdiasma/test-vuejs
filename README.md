Movie App

Application web pour gérer des films, avec un backend NestJS et un frontend Vue.js utilisant Vuetify et Pinia.

****Cloner le dépôt :
-git clone <ton-repo-url>
-cd <nom-du-projet>


****Installer les dépendances

Backend (NestJS)
-cd backend
-npm install

Frontend (Vue.js)
-cd frontend
-npm install



****Lancer le projet locale(dev)
Backend NestJS
-cd backend
-npm run start:dev  
-Port : 3000

Frontend Vue.js
-cd frontend
-npm run start:dev
-Port 8080


****Architecture

backend/ → NestJS API

frontend/ → Vue 3 + Vuetify + Pinia
